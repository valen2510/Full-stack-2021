# Full stack Open Course 2021

## Description

The main focus of this course is on building single page applications with ReactJS that use REST APIs built with Node.js. The course also contains a section on GraphQL, a modern alternative to REST APIs. The course also covers testing, configuration and environment management, and the use of MongoDB for storing the applicationâ€™s data.
